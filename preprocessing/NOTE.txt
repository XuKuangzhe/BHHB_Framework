================================================================================
NOTE ON RAW DATA AVAILABILITY AND PREPROCESSING
================================================================================

1. Raw Data Exclusion
---------------------
The raw data files generated by PsychoPy and the Tobii Eye Tracker (in .tsv and .csv formats) 
are NOT included in this repository. This decision was made for the following reasons:

    a) File Size: The raw high-frequency gaze data (120Hz) for 84 participants across 
       multiple trials amounts to several gigabytes, exceeding standard repository limits.
    
    b) Privacy: Raw data files contain timestamps and metadata that could potentially 
       compromise participant anonymity if combined with external information.

2. Preprocessing Logic (Demonstrated in 00_Raw_to_Clean.R)
---------------------
The script `00_Raw_to_Clean.R` is provided in this folder to demonstrate the exact 
logic used to transform the raw data into the analysis-ready dataset (`sumGDT.csv`). 

Key steps documented in the script include:
    - Synchronization: Aligning PsychoPy stimulus timestamps with Tobii gaze timestamps 
      using a linear model (lmv).
    - Event Detection: Filtering gaze points based on Tobii Validity Codes (< 3).
    - Coordinate Transformation: Mapping raw coordinates to screen resolution (1920x1080).
    - Heatmap Generation: Applying Gaussian smoothing (sigma=10) to discrete fixation/mouse points.
    - AOI Weighting: Calculating density scores for specific facial features (e.g., Eyes, Mouth) 
      using predefined masks.

3. Reproducibility
---------------------
The provided dataset `data/sumGDT.csv` is the direct output of this preprocessing pipeline. 
All subsequent analyses (ZIB modeling, etc.) in the `analysis/` folder use `sumGDT.csv` 
as the starting point.

If access to the raw data is strictly necessary for verification purposes, please contact 
the corresponding author directly.
